# 📝 Catatang

**Catatang** ialah aplikasi nota ringkas berasaskan web dengan reka bentuk **liquid glass style** seperti UI Apple.
Ia membolehkan pengguna menambah dan memadam nota dengan antaramuka moden dan animasi yang lembut.

---

## ✨ Ciri-ciri

* ➕ Tambah nota
* 🗑 Buang nota
* 🌙 Mod gelap (Dark Mode)
* 🔔 Toast notification
* ⏱ Paparan masa nota ditambah
* 💧 Liquid glass UI design
* ⚡ Animasi spring yang smooth

---

## 🖼 Preview

Contoh paparan aplikasi:

```
Catatang
Nota untuk semua orang

[Tulis nota...]
[Tambah]

• Nota pertama
• Nota kedua
```

## 🚀 Cara Jalankan Projek

### 1. Clone repository

```
git clone https://github.com/username/catatang.git
```

### 2. Masuk folder projek

```
cd catatang
```

### 3. Jalankan secara local

Buka sahaja:

```
index.html
```

atau gunakan Netlify CLI:

```
netlify dev
```
